# 功能说明（Features）

更新日期：2026-05-06

## 下载能力

- HTTP/HTTPS 直链下载
- M3U8/HLS 下载
- YouTube 信息与格式相关能力
- 可配置并发数量
- 断点续传、暂停、恢复、失败重试
- 速率、进度、ETA、完成统计

## 批量导入

- CSV / Excel 导入
- 导入预览和字段映射
- 编码检测
- 无效行摘要
- 重复导入识别：新增、已存在、已完成、可续传、等待、失败任务分组

## 任务管理

- 添加、删除、开始、暂停、恢复、取消
- 全部开始、全部暂停、失败重试、清理完成任务
- 保存位置确认和批量输出目录调整
- App 重启后恢复任务和本地断点状态，但不自动继续下载

## 可靠性

- `.part` 临时文件
- resume 快照
- `.vdstate` 完成 marker
- 下载错误分类和诊断提示
- 后端状态为真相源，前端通过事件和 refresh 同步

## 可观察性

- `download-events` 事件流
- 前端任务状态和统计面板
- 本地测试包可落地后端/前端日志
- Graphify 架构图谱和 GitNexus 影响分析辅助维护
